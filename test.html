🔗 Fixing CORS Errors

Best option — Paste M3U:
Switch to 📋 Paste M3U tab, open your .m3u file in Notepad, copy all the text and paste it there. No CORS issues.

Option 2 — Wrap URL with proxy:
Prefix your URL like this:

https://corsproxy.io/?YOUR_PLAYLIST_URL

Option 3 — Run locally:
Open this HTML file from a local server (VS Code Live Server etc.) and use a relative path like playlist.m3u
