Comprehensive Bug Classes, Tools, and Workshops

Overview of the This page

  • Expands Bug Hunters Methodology with an exhaustive list of vulnerabilities by severity.
  • Provides comprehensive tools for recon, fuzzing, and exploitation across heat-mapped areas.
  • Includes workshops targeting low-to-critical bugs in enterprise apps.
  • Bridges beginner-to-advanced skills for bug bounty hunters.

Vulnerability Classes by Severity

Critical Severity

Remote Code Execution (RCE)

  • Command injection or deserialization in admin panels/uploads.
  • Impact: Full server compromise.
  • Tools: Commix, Burp Suite.
  • commix -u https://target.com/admin?cmd=whoami

SQL Injection (SQLi)

  • Target forms, APIs with parameters like id, search.
  • Impact: Database access, data theft, admin takeover.
  • Tools: sqlmap, Burp Suite.
  • sqlmap -u https://target.com/search?query=1 --dbs

Command Injection

  • Inject commands in admin panels or API inputs.
  • Impact: Execute arbitrary system commands.
  • Tools: Commix, Burp Suite.
  • commix -u https://target.com/admin?cmd=whoami

Authentication Bypass

  • Bypass login via weak auth checks or token manipulation.
  • Impact: Unauthorized access to accounts.
  • Tools: Burp Suite, jwt_tool.
  • jwt_tool -t token -p

Insecure Deserialization

  • Untrusted data in APIs/forms leading to object injection.
  • Impact: Code execution, data manipulation.
  • Tools: ysoserial, Burp Suite.
  • ysoserial -g ObjectPayload -o raw > payload.bin

Privilege Escalation

  • Escalate from user to admin via misconfigured roles.
  • Impact: Admin-level access.
  • Tools: Burp Suite, Postman.
  • Use Burp Repeater to test role-based access

Buffer Overflow

  • Overflow input buffers in APIs or forms.
  • Impact: Code execution or crashes.
  • Tools: Burp Suite, FuzzDB.
  • Use Burp Intruder with FuzzDB payloads

Path Traversal

  • Access restricted files via file parameters (e.g., ../../etc/passwd).
  • Impact: Sensitive file access.
  • Tools: dotdotpwn, Burp Suite.
  • dotdotpwn -h https://target.com/file?path=../../etc/passwd

Broken Access Control

  • Unprotected APIs/admin endpoints.
  • Impact: Admin function/data access.
  • Tools: Postman, Arjun.
  • arjun -u https://target.com/api/v1

Unrestricted File Upload

  • Upload executable files in misconfigured servers.
  • Impact: Server shelling, RCE.
  • Tools: Burp Suite, Nuclei.
  • nuclei -u https://target.com/upload -t file-upload/

Use After Free

  • Exploit freed memory references in app logic.
  • Impact: Code execution or crashes.
  • Tools: Burp Suite, custom fuzzers.
  • Use Burp Intruder with custom payloads

Stack Overflow / Heap Overflow

  • Overflow stack/heap in input handling.
  • Impact: Code execution or crashes.
  • Tools: Burp Suite, FuzzDB.
  • Use Burp Intruder with FuzzDB payloads

Format String Vulnerabilities

  • Exploit format string inputs in APIs.
  • Impact: Memory corruption, code execution.
  • Tools: Burp Suite, custom fuzzers.
  • Use Burp Intruder with format string payloads

Cross-Site Request Forgery (CSRF) – Critical Cases

  • CSRF in critical functions (e.g., admin actions).
  • Impact: Unauthorized critical actions.
  • Tools: Burp Bounty, wfuzz.
  • wfuzz -u https://target.com/admin -d "action=FUZZ" -w csrf.txt

Cross-Site Scripting (XSS) – Stored/DOM

  • Persistent or DOM-based XSS in profiles.
  • Impact: Session hijacking, data theft.
  • Tools: Burp Bounty, XSStrike.
  • xsstrike -u "https://target.com/profile?bio=<script>alert(1)</script>"

Insecure Default Configuration

  • Default admin accounts or settings.
  • Impact: Unauthorized access.
  • Tools: Nuclei, Nmap.
  • nuclei -u https://target.com -t misconfig/

Exposed Administrative Interfaces

  • Unprotected admin panels (e.g., /admin).
  • Impact: Full app control.
  • Tools: ffuf, Burp Suite.
  • ffuf -u https://target.com/FUZZ -w admin.txt

Weak Cryptography

  • Hardcoded keys, weak algorithms (e.g., MD5, RC4).
  • Impact: Decryption of sensitive data.
  • Tools: MobSF, APKLeaks.
  • apkleaks -f app.apk

High Severity

XML External Entity (XXE)

  • XML uploads/APIs with entity parsing.
  • Impact: File access, internal service interaction.
  • Tools: Burp Suite, Nuclei.
  • curl -X POST https://target.com/xml -H "Content-Type: application/xml" -d ''

Server-Side Request Forgery (SSRF)

  • Redirect URLs/APIs accessing internal services.
  • Impact: Internal network access.
  • Tools: Burp Suite, ffuf.
  • curl -X GET https://target.com/redirect?url=http://169.254.169.254

Unvalidated Redirects and Forwards

  • Unvalidated redirect URLs.
  • Impact: Phishing or SSRF setup.
  • Tools: ffuf, Burp Suite.
  • ffuf -u https://target.com/redirect?url=FUZZ -w urls.txt

Improper Session Handling

  • Weak session IDs or non-regenerated sessions.
  • Impact: Session hijacking.
  • Tools: Burp Suite (Sequencer).
  • Use Burp Sequencer to analyze session tokens

Mass Assignment Vulnerabilities

  • APIs allowing unauthorized parameter updates.
  • Impact: Privilege escalation, data tampering.
  • Tools: Postman, Burp Suite.
  • Use Burp Repeater to test parameter updates

Race Conditions

  • Concurrent requests bypassing checks (e.g., payments).
  • Impact: Financial loss, unauthorized actions.
  • Tools: Burp Suite (Turbo Intruder).
  • Use Turbo Intruder for concurrent requests

Insecure Permissions on Files/Services

  • Misconfigured file or service permissions.
  • Impact: Unauthorized access.
  • Tools: Nuclei, Nmap.
  • nmap -sV -p- target.com

Failure to Rotate Secrets or Keys

  • Static API keys or credentials.
  • Impact: Long-term unauthorized access.
  • Tools: APKLeaks, Burp Suite.
  • apkleaks -f app.apk

Insecure CORS Configuration

  • Allowing unauthorized origins in CORS.
  • Impact: Unauthorized API access.
  • Tools: CORScanner, Burp Suite.
  • corscanner -u https://target.com

Misconfigured Cloud Storage

  • Open S3 Bugets or cloud storage.
  • Impact: Data exposure.
  • Tools: CloudSploit, Amass.
  • amass enum -d target.com

Improper Certificate Validation

  • Weak SSL/TLS validation.
  • Impact: MITM attacks.
  • Tools: sslscan, Burp Suite.
  • sslscan target.com

DNS Rebinding

  • Exploit DNS rebinding for internal access.
  • Impact: Bypassing network restrictions.
  • Tools: Burp Suite, custom scripts.
  • Use custom DNS rebinding tool

HTTP Request Smuggling

  • Exploit HTTP request parsing mismatches.
  • Impact: Bypassing security controls.
  • Tools: Burp Suite, HTTP Smuggler.
  • Use HTTP Smuggler Burp extension

Improper Input Validation

  • Weak input sanitization in forms/APIs.
  • Impact: Enables multiple attack vectors.
  • Tools: Burp Suite, FuzzDB.
  • Use Burp Intruder with FuzzDB payloads

Medium Severity

Information Disclosure

  • Exposed data in errors, stack traces, logs.
  • Impact: Aids reconnaissance.
  • Tools: Burp Suite, WayMore.
  • waymore -u https://target.com

Clickjacking

  • Missing X-Frame-Options allowing UI redressing.
  • Impact: Tricking users into actions.
  • Tools: Burp Suite, httpx.
  • httpx -u https://target.com -headers

Directory Listing Enabled

  • Exposed directories with sensitive files.
  • Impact: File exposure.
  • Tools: gobuster, Feroxbuster.
  • gobuster dir -u https://target.com -w dirlist.txt

Verbose Server Banners

  • Server version leaks in headers.
  • Impact: Aids targeted attacks.
  • Tools: httpx, Nmap.
  • httpx -u https://target.com -headers

Outdated Software with Known Vulnerabilities

  • Vulnerable libraries or server software.
  • Impact: Known exploit execution.
  • Tools: Retire.js, Dependency-Check.
  • retire --js https://target.com

Open Redirects

  • Unvalidated redirect URLs.
  • Impact: Phishing or SSRF setup.
  • Tools: ffuf, Burp Suite.
  • ffuf -u https://target.com/redirect?url=FUZZ -w urls.txt

Insecure Cookie Flags

  • Missing HttpOnly, Secure, SameSite flags.
  • Impact: Session hijacking risk.
  • Tools: Burp Suite, httpx.
  • httpx -u https://target.com -cookies

Sensitive Data in URLs or Logs

  • Tokens or PII in URLs/logs.
  • Impact: Data exposure.
  • Tools: Burp Suite, WayMore.
  • waymore -u https://target.com

Improper Asset Management

  • Untracked subdomains or assets.
  • Impact: Hidden attack surfaces.
  • Tools: Amass, Subadub.
  • amass enum -d target.com

Client-Side Validation Only

  • Validation bypass via direct requests.
  • Impact: Unauthorized actions.
  • Tools: Burp Suite, Postman.
  • Use Burp Repeater to bypass validation

Lack of Rate Limiting

  • No limits on API/login requests.
  • Impact: Brute-forcing, DoS.
  • Tools: Burp Suite, Hydra.
  • hydra -l user -P passlist.txt target.com http-post-form

Improper Use of JWT / Token Expiry

  • Non-expiring or weak JWTs.
  • Impact: Unauthorized access.
  • Tools: jwt_tool, Burp Suite.
  • jwt_tool -t token -p

No CAPTCHA or Brute-force Protection

  • Missing CAPTCHA on login forms.
  • Impact: Account brute-forcing.
  • Tools: Burp Suite, Hydra.
  • hydra -l user -P passlist.txt target.com http-post-form

Low Severity

Autocomplete Enabled on Sensitive Fields

  • Autocomplete on password fields.
  • Impact: Credential exposure on shared devices.
  • Tools: Burp Suite, httpx.
  • httpx -u https://target.com -forms

Email Enumeration

  • APIs revealing valid emails.
  • Impact: Aids phishing campaigns.
  • Tools: Burp Suite, Postman.
  • Use Burp Repeater to test email endpoints

Click Tracking Vulnerabilities

  • Trackable user actions in analytics.
  • Impact: Privacy leakage.
  • Tools: Burp Suite, FuzzDB.
  • Use Burp Intruder with tracking payloads

Deprecated HTML Tags or Attributes

  • Outdated HTML increasing attack surface.
  • Impact: Enables legacy exploits.
  • Tools: Burp Suite, Wappalyzer.
  • wappalyzer https://target.com

Security Headers Missing

  • Missing CSP, X-Content-Type-Options.
  • Impact: Enables XSS, MIME-type attacks.
  • Tools: httpx, Burp Suite.
  • httpx -u https://target.com -headers

Insecure API Response Codes

  • Verbose API error codes.
  • Impact: Information leakage.
  • Tools: Postman, Burp Suite.
  • Use Postman to test API responses

Missing SPF/DKIM/DMARC for Emails

  • Weak email authentication.
  • Impact: Email spoofing.
  • Tools: dmarcian, custom scripts.
  • Use dmarcian to check SPF/DKIM

Potential Debug Information Leaked

  • Debug data in responses.
  • Impact: Aids reconnaissance.
  • Tools: Burp Suite, WayMore.
  • waymore -u https://target.com

Exposure of Internal IPs

  • Internal IPs in headers or errors.
  • Impact: Network mapping aid.
  • Tools: Burp Suite, httpx.
  • httpx -u https://target.com -headers

Autocomplete Password Fields

  • Autocomplete on password inputs.
  • Impact: Credential exposure.
  • Tools: Burp Suite, httpx.
  • httpx -u https://target.com -forms

Mixed Content (HTTP over HTTPS)

  • HTTP resources on HTTPS pages.
  • Impact: MITM risk.
  • Tools: Burp Suite, httpx.
  • httpx -u https://target.com -mixed-content

Special Cases

Zero-Day Vulnerabilities

  • Unknown vulnerabilities in apps.
  • Impact: Varies, potentially critical.
  • Tools: Custom fuzzers, Burp Suite.
  • Use Burp Intruder with custom payloads

N-Day Vulnerabilities

  • Known but unpatched CVEs.
  • Impact: Known exploit execution.
  • Tools: Nuclei, Metasploit.
  • nuclei -u https://target.com -t cves/

Social Engineering Vectors

  • Phishing via app weaknesses.
  • Impact: User credential theft.
  • Tools: Burp Suite, Postman.
  • Use Burp Repeater to test phishing endpoints

Supply Chain Attacks

  • Vulnerable third-party libraries.
  • Impact: Compromise via dependencies.
  • Tools: Retire.js, Dependency-Check.
  • dependency-check --scan target.com

Typosquatting and Dependency Confusion

  • Malicious packages mimicking legit ones.
  • Impact: Code execution via dependencies.
  • Tools: Dependency-Check, Amass.
  • dependency-check --scan target.com

Man-in-the-Middle (MITM) Weaknesses

  • Weak TLS or certificate issues.
  • Impact: Data interception.
  • Tools: sslscan, Burp Suite.
  • sslscan target.com

Side-channel Attacks

  • Spectre, Meltdown, or Rowhammer exploits.
  • Impact: Data leakage via hardware.
  • Tools: Custom scripts, manual analysis.
  • Use custom side-channel test scripts

Logic Flaws / Business Logic Errors

  • Exploiting app logic (e.g., price manipulation).
  • Impact: Financial loss, unauthorized actions.
  • Tools: Burp Suite, Postman.
  • Use Burp Repeater to manipulate requests

Physical Security Weaknesses

  • Physical access to servers or devices.
  • Impact: Direct system compromise.
  • Tools: Manual analysis, OSINT.
  • Use OSINT for physical recon

Privilege Misconfigurations

  • Misconfigured IAM roles or sudo access.
  • Impact: Privilege escalation.
  • Tools: CloudSploit, Burp Suite.
  • cloudsploit scan --target target.com

DLL Hijacking / Library Injection

  • Inject malicious DLLs via app flaws.
  • Impact: Code execution.
  • Tools: MobSF, custom scripts.
  • mobsf -f app.exe

API Abuse / Rate Limit Evasion

  • Bypassing API rate limits.
  • Impact: DoS or data scraping.
  • Tools: Burp Suite, Postman.
  • Use Burp Intruder for rate limit testing

Security Through Obscurity

  • Reliance on hidden endpoints.
  • Impact: False sense of security.
  • Tools: ffuf, WayMore.
  • ffuf -u https://target.com/FUZZ -w hidden.txt

Comprehensive Toolset

Reconnaissance and Fingerprinting

  • Subadub: Fast subdomain enumeration.
  • subadub -d target.com
  • httpx: HTTP probing with tech detection.
  • httpx -l subdomains.txt -status-code -tech-detect
  • Amass: Comprehensive asset discovery.
  • amass enum -d target.com
  • WhatRuns/Wappalyzer: Tech stack fingerprinting.
  • wappalyzer https://target.com
  • CloudSploit: Cloud misconfiguration scanning.
  • cloudsploit scan --target target.com

Port Scanning

  • RustScan: High-speed port scanning.
  • rustscan -a target.com --ports 1-65535
  • Nabu: Lightweight port scanner.
  • nabu -t target.com
  • Nmap: Detailed scanning with scripts.
  • nmap -sV -p- target.com

Workshops

  • Practical Bug Hunting Lab:
    • Test low-to-critical vulns (CSRF, SQLi, RCE, XXE) in heat-mapped areas.
    • Use OWASP Vulnerable Web Apps for practice.
    • Configure Burp Suite, Nuclei, XSStrike, sqlmap, Commix, jwt_tool.
  • API Hacking Workshop:
    • Target APIs for broken access control, key leakage, OAuth, JWT flaws.
    • Use Arjun, Postman, MobSF, APKLeaks, jwt_tool.
    • arjun -u https://target.com/api/v1
  • Advanced JavaScript Analysis:
    • Parse JS with JS-Miner, DeobfuscateJS, GAP for XSS, IDOR.
    • Test account/profile sections for endpoint leaks.
    • js-miner -i https://target.com/app.js
  • Automation Pipeline Building:
    • Build pipelines with ReconFTW, httpx, Nuclei, Notify.
    • Monitor with changedetection.io for new vulns.
    • reconftw -d target.com --all | notify -provider slack
  • Reporting and Triage Workshop:
    • Write reports for critical vulns (RCE, SQLi, XXE, privilege escalation).
    • Practice triage on HackerOne/Bugcrowd.
    • Mock submissions for XSS, IDOR, OAuth, JWT flaws.

Integration with Bug Hunters Methodology and Heat Map Areas

  • Apply vulnerabilities to heat-mapped areas with appropriate tools:
    • Upload Functions:
      • Vulns: Unrestricted File Upload, XSS, XXE, SSRF, Server Shelling, RCE, File Inclusion, Buffer Overflow.
      • Tools: Burp Suite, Nuclei, Commix, dotdotpwn, FuzzDB.
      • nuclei -u https://target.com/upload -t file-upload/
    • Multi-Part Forms:
      • Vulns: SQLi, XSS, CSRF, SSTI, Business Logic Flaws, Mass Assignment, Race Conditions, Client-Side Validation Only.
      • Tools: Arjun, Burp Bounty, sqlmap, XSStrike, tplmap, Turbo Intruder.
      • arjun -u https://target.com/form
    • Content Types (XML/JSON):
      • Vulns: XXE, JSON Injection, Information Disclosure, Insecure Deserialization, Improper Input Validation.
      • Tools: Burp Suite, Nuclei, ysoserial.
      • nuclei -u https://target.com -t xxe/
    • APIs:
      • Vulns: Broken Access Control, API Key Leakage, IDOR, SQLi, JWT Manipulation, OAuth Misconfiguration, API Abuse, Insecure CORS, HTTP Request Smuggling.
      • Tools: Postman, Arjun, APKLeaks, jwt_tool, CORScanner, HTTP Smuggler.
      • arjun -u https://target.com/api/v1
    • Account/Profile Sections:
      • Vulns: Stored XSS, SSTI, IDOR, Information Disclosure, Privilege Escalation, Authentication Bypass, Weak Cryptography, Session Handling.
      • Tools: Burp Bounty, JS-Miner, Hunt, tplmap, jwt_tool.
      • hunt -u https://target.com/profile
    • Third-Party Integrations:
      • Vulns: Supply Chain Attacks, Outdated Libraries, Weak Cryptography, Insecure CORS, Misconfigured Cloud Storage.
      • Tools: Retire.js, MobSF, Dependency-Check, CloudSploit.
      • retire --js https://target.com
    • URLs/Redirect Parameters:
      • Vulns: SSRF, Open Redirect, Path Traversal, File Inclusion, DNS Rebinding, HTTP Request Smuggling.
      • Tools: Burp Suite, ffuf, dotdotpwn.
      • ffuf -u https://target.com/redirect?url=FUZZ -w urls.txt
    • Errors:
      • Vulns: SQLi, Path Traversal, Information Disclosure, Crash Vectors, Verbose Server Banners, Potential Debug Information.
      • Tools: sqlmap, Burp Suite, Nuclei, WayMore.
      • sqlmap -u https://target.com/error?input=1
    • General System:
      • Vulns: Zero-Day, N-Day, Insecure Default Configuration, Exposed Admin Interfaces, Privilege Misconfigurations, Side-channel Attacks, DLL Hijacking, Security Through Obscurity.
      • Tools: Nuclei, Metasploit, CloudSploit, MobSF, custom scripts.
      • nuclei -u https://target.com -t misconfig/
  • Use tools across recon, enumeration, and exploitation for efficiency.
  • Workshops reinforce testing in heat-mapped areas with low-to-critical vulns.
  • Maintain mental resilience: Every app has exploitable bugs.
