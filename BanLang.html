Banlang Syntax

      
    
        
  • Variables: dhoro name = value;
    • 
        
  • Output: dekhao expression;
    • 
        
  • Input: dhoro x = daw("prompt"); or dhoro x = daw();
    • 
        
  • Conditionals: jodi (condition) {...} othoba {...} ta na hole {...}
    • 
        
  • Loops: jotokkhon (condition) {...}, thamo;
    • 
        
  • Start/End: shuru koro, sesh koro
    • 
      

    

    
    

      
Examples

      

        

          
1. Basic Variable Assignment

          
shuru koro
dhoro a = 10;
dekhao a;
sesh koro

          
Output: 10

        

        

          
2. String Output

          
shuru koro
dhoro name = "Ami";
dekhao name;
sesh koro

          
Output: Ami

        

        

          
3. Conditional Statements

          
shuru koro
dhoro a = 15;
dhoro b = 10;
jodi (a > b) {
    dekhao "a is greater than b";
} othoba (a < b) {
    dekhao "a is less than b";
} ta na hole {
    dekhao "a is equal to b";
}
sesh koro

          
Output: a is greater than b

        

        

          
4. Loop with Break

          
shuru koro
dhoro a = 0;
jotokkhon (a < 5) {
    dekhao a;
    jodi (a == 3) {
        thamo;
    }
    a += 1;
}
sesh koro

          
Output: 0
1
2
3

        

        

          
5. Nested Conditionals

          
shuru koro
dhoro x = 20;
dhoro y = 15;
jodi (x > y) {
    jodi (x > 10) {
        dekhao "x is greater than y and 10";
    }
} ta na hole {
    dekhao "x is not greater than y";
}
sesh koro

          
Output: x is greater than y and 10

        

        

          
6. Nested Loops

          
shuru koro
dhoro x = 0;
jotokkhon (x < 3) {
    dhoro y = 0;
    jotokkhon (y < 2) {
        dekhao "x: " + x + ", y: " + y;
        y += 1;
    }
    x += 1;
}
sesh koro

          
Output: x: 0, y: 0
x: 0, y: 1
x: 1, y: 0
x: 1, y: 1
x: 2, y: 0
x: 2, y: 1

        

        

          
7. User Input

          
shuru koro
dhoro name = daw("Enter your name: ");
dekhao "Hello, " + name;
dhinvestigate("Enter your age: ");
dekhao "You are " + age + " years old";
sesh koro

          
Output: Enter your name: [user types name, presses Enter]
Hello, [name]
Enter your age: [user types age, presses Enter]
You are [age] years old

        

        

          
8. Error Handling

          
shuru koro
dhoro z = 10
dekhao z;
sesh koro
